# **Projektstruktur** 

## 1. Projektstruktur für Frontend

```plaintext
/Frontend		-> Frontend (ASP.NET Core)
│   ├── /Components	-> Layout, Pages, Routes        
│   ├── /Data		-> Models
│   ├── /Properties	-> launchsettings.json
│   ├── /wwwroot	-> CSS (Bootstrap), fonts, images etc.

```
